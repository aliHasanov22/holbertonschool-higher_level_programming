 # 🚀 Higher Level Programming – Holberton School

This repository contains a comprehensive collection of projects completed as part of the **Holberton School Higher-Level Programming curriculum**. It focuses on mastering **Python, SQL, APIs, and backend development concepts**, building a strong foundation for software engineering and cybersecurity paths.

---

## 📌 Overview

This project is designed to develop advanced programming skills using:

- 🐍 **Python** (core language & OOP)
- 🗄️ **SQL** (database design & queries)
- 🌐 **RESTful APIs**
- 🔐 **Backend fundamentals**
- 📡 **Networking & data handling**

It progresses from basic syntax to real-world backend systems.

## ⚙️ Requirements

- **OS:** Ubuntu 20.04 LTS  
- **Python:** 3.8+  
- **Style:** `pycodestyle` (PEP8 compliant)  
- All files must:
  - End with a new line
  - Start with `#!/usr/bin/env python3`
  - Be executable

## ▶️ How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/holbertonschool-higher_level_programming.git
cd holbertonschool-higher_level_programming
```

Run Python files:

```bash
./script.py
```

Or:

```bash
python3 script.py
```
